## Project Overview

-   **Project Name**: Rentify
-   **Framework**: CakePHP 5
-   **Purpose**: Car rental management system
-   **Local URL**: http://localhost/rentify

---

## Key Information

### Database

-   Database name: `rentify`
-   Main tables: `users`, `cars`, `bookings`, `payments`, `reviews`, `car_categories`, `maintenance`,

### User Roles

-   **Admin**: Full access to dashboard and management features
-   **Customer**: Can browse cars, make bookings, leave reviews

### Important Directories

-   `templates/` - View templates
-   `src/Controller/` - Controllers
-   `src/Model/` - Models (Table and Entity classes)
-   `webroot/` - Public assets (CSS, JS, images)
